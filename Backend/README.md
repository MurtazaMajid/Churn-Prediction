# Campbell's AI Marketing — FastAPI Backend

## Setup

```bash
pip install -r requirements.txt
```

## Environment Variables

Set these before running:

```bash
export GROQ_API_KEY="your_groq_key_here"
export DATA_DIR="."   # folder containing CSV + pickle files
```

## Run Locally

```bash
uvicorn main:app --reload --port 8000
```

Open docs at: http://localhost:8000/docs

## Deploy on Railway

1. Push this folder to GitHub
2. Go to railway.app → New Project → Deploy from GitHub
3. Add environment variables: `GROQ_API_KEY`, `DATA_DIR`
4. Railway auto-detects FastAPI and deploys

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | `/` | Health check |
| POST | `/api/segment` | Customer segment (New/Occasional/Regular/Lost) |
| POST | `/api/churn` | Churn probability + risk level |
| POST | `/api/sentiment` | ABSA triplets from review text |
| POST | `/api/generate-message` | Personalized SMS + email + app notification |
| POST | `/api/full-pipeline` | All of the above in one call |
| GET | `/api/dashboard` | Aggregated stats for dashboard |

## Example Request

```bash
curl -X POST http://localhost:8000/api/full-pipeline \
  -H "Content-Type: application/json" \
  -d '{
    "recency": 89,
    "frequency": 2,
    "monetary": 266.17,
    "favorite_items": ["Slider Flight", "Taco Flight"],
    "review": "The food was amazing but service was really slow"
  }'
```
